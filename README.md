# Intune Device Cleanup Rule Automation

> Reads, reports on, and optionally configures Intune device cleanup rules via the Microsoft Graph API, including a stale device summary across configurable inactivity thresholds.

## Overview

This PowerShell project was extracted from an article on [MSEndpoint.com](https://msendpoint.com) and is part of the professional automation portfolio.

### Quick Start

```powershell
# Clone the repository
git clone https://github.com/Msendpoint/intune-device-cleanup-rule-automation.git
cd intune-device-cleanup-rule-automation

# One-click install & run (checks prerequisites, installs modules, launches main script)
.\Install.ps1
```

## Tech Stack

PowerShell, Microsoft Graph

## Requirements

- PowerShell 7.0 or later
- Microsoft Graph CLI (if applicable)
- Appropriate administrative permissions
